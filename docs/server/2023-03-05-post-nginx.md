# Nginx란?
클라이언트와 직접 연결되는 서버로서, 전달받은 요청을 적절한 작업 서버에 분배해주고 결과를 클라이언트에 전달해주는 중간 역할 서버입니다. 이를 이용하여 로드 밸런서로서의 역할도 하게 됩니다.
